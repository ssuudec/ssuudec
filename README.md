## Featured Projects

### [TravelBuddy Mobile App](https://github.com/ssuudec/TravelBuddy-MobileApp)

A mobile travel planning and route management app built for organizing trips with real-time group communication.

- Developed with React Native and React Navigation.
- Uses Firebase Authentication and Firestore for user data and real-time group chats.
- Includes Google Places API integration, dynamic city images and OpenWeatherMap weather data.
- Focuses on mobile UX, trip planning, profile management and date conflict prevention.

### [ChemWaste](https://github.com/ssuudec/ChemWaste)

A chemical waste management and tracking system developed as a web programming course project.

- Built with Vanilla PHP, MySQL/MariaDB, Bootstrap 5, HTML and CSS.
- Includes user registration/login, session management and password hashing with bcrypt.
- Supports CRUD operations for chemical waste records.
- Tracks risk levels, storage status and user-specific waste inventory.
- Designed with a responsive interface for practical laboratory waste tracking.

### [Implant Surgery Database System](https://github.com/ssuudec/implant-surgery-database-system)

Relational database design and SQL implementation for a dental implant surgery and patient tracking system.

- Designed a healthcare-focused database model for patients, doctors, appointments, implant surgeries and post-surgery controls.
- Includes ER diagrams, relational schema, SQL table definitions, sample data and queries.
- Built with MySQL and MySQL Workbench.
- Shows strong understanding of database modeling, requirements analysis and structured data design.

### [Crazy Neighbors](https://github.com/ssuudec/crazy-neighbors)

A turn-based 2D shooting game with a humorous Turkish neighborhood atmosphere.

- Developed with Unity and C#.
- Includes mechanics such as aiming, shooting, shielding, double shots and morale-based actions.
- Published as a WebGL game on itch.io.
- Combines game logic, UI interaction and player-centered gameplay design.

### [Mağaradan Kaçış](https://github.com/ssuudec/magaradan-kacis-game)

A browser-based puzzle/platform game inspired by recursive room mechanics.

- Built with HTML5 Canvas, Vanilla JavaScript and CSS.
- Features multiple levels with changing rules, movement mechanics, reset controls and hints.
- Includes custom game flow, level logic, sound settings and visual assets.
- Shows creativity in web-based game development without relying on external game libraries.

### [Turkish NLP Chunking Project](https://github.com/ssuudec/nlp-project)

A Natural Language Processing project for detecting Turkish sentence structures.

- Uses BIO tagging and CRF models.
- Detects CHUNK-OUTER, CHUNK-INNER and CLAUSE structures.
- Includes data generation, model training, evaluation and prediction scripts.
- Demonstrates interest in NLP, model pipelines and language-focused machine learning.

## Other Projects

### [Hamming Code Simulator](https://github.com/ssuudec/hamming-code-simulator)

Interactive web simulator for Hamming SEC-DED error detection and correction.

- Built with React, TypeScript and Tailwind CSS.
- Supports 8-bit and 16-bit data input.
- Simulates parity bit calculation, memory storage, error injection and correction.

### [Fuzzy Logic ALE Prediction](https://github.com/ssuudec/fuzzy-logic)

Fuzzy logic system for estimating Average Localization Error in wireless sensor networks.

- Built with Python and scikit-fuzzy.
- Compares membership functions and defuzzification methods.
- Includes MAE and RMSE-based evaluation.

### [Convex Hull Analysis](https://github.com/ssuudec/convex-hull-analysis)

Algorithm analysis project comparing Brute Force and Graham Scan approaches.

- Built with Python, Tkinter and Matplotlib.
- Visualizes convex hull algorithms and compares performance.
- Focuses on algorithm complexity, visualization and experimental analysis.
